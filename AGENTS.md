# Указания для ИИ (Agents)

## Основной репозиторий-образец

**Используй как главный референс при создании новых алгоритмов и структур данных:**

- **Репозиторий:** [trekhleb/javascript-algorithms](https://github.com/trekhleb/javascript-algorithms)
- **Назначение:** образец структуры папок, стиля кода, именования и организации тестов.

При добавлении новых файлов в этот проект опирайся на соответствующий раздел в том репозитории (например, `src/algorithms/sorting/quick-sort/`, `src/algorithms/search/binary-search/`, `src/algorithms/uncategorized/recursive-staircase/` и т.д.).

## Структура проекта

- `src/algorithms/` — алгоритмы по категориям (sorting, search, sets, string, graph, math, uncategorized и др.).
- `src/data-structures/` — структуры данных (linked-list, stack, queue, hash-table, tree и др.).
- `src/playground/` — песочница для экспериментов.
- `src/utils/` — вспомогательные функции.

Задачи осваиваем **поочерёдно**; при создании новой реализации сохраняй единообразие со структурой и стилем репозитория-образца.
